# Bhavesh Meghwal

Mechanical Engineering student at IIT (BHU) Varanasi, building around aerial robotics, AI tooling, and product systems.

Most of my work starts with drones, control, or operator workflows. Then it spills into software. I like systems that have to hold up in the real world, whether that means PX4 fault-tolerant control, mission-planning tools, or AI products built on top of field data.

## What I work on

- PX4, ArduPilot, ROS, and drone autonomy
- Operator-facing tools for mission planning and flight analysis
- AI products that turn raw technical data into usable workflows
- Product systems across frontend, backend, and infra boundaries

## Current work

- Building flight-log analysis tools for drone operators
- Working on mission-planning and fleet-operations software
